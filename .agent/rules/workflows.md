# Complex Task Workflow

## Protocol for Large Requirements

When handling complex user requests, follow this strict workflow:

1.  **Plan**: Create a `plan.md` (or `implementation_plan.md`) to document the strategy.
    *   **Language**: English or Spanish (per user preference), but strictly **English** for technical details (code blocks, logic).
2.  **Structure**: Define data interfaces first.
    *   **Language**: TypeScript interfaces must be in **English**.
3.  **Implementation**: Write iterative code.
4.  **Review**: Self-verify before finishing:
    *   Are variables in English?
    *   Is the UI text in Spanish?
